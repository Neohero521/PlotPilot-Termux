├── Menu.sh         - 管理菜单

└── stop.sh         - 停止服务

## 使用方法

### 一键安装

```bash
curl -O https://raw.githubusercontent.com/Neohero521/PlotPilot-Termux/main/Install.sh && bash Install.sh
```

### 手动安装

1. 克隆仓库
```bash
git clone https://github.com/Neohero521/PlotPilot-Termux.git
cd PlotPilot-Termux
```

2. 运行安装
```bash
bash Install.sh
```
